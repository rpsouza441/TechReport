# Plan 01: Sync Queue Bug Fixes

## Objective

Corrigir 3 bugs na fila de sincronizacao:
1. RAT fica pendente apos falha offline
2. Botao retry nao funciona
3. Fila nao mostra ID da RAT

## Tasks

### Task 1: Investigar codigo de sync

**Files to read:**
- `lib/features/rat/domain/services/rat_sync_coordinator.dart`
- `lib/features/sync/data/repositories/sync_queue_repository.dart`
- `lib/shared/presentation/screens/sync_queue_screen.dart`

**Action:**
1. Identificar onde a fila de sync e inicializada ao iniciar app
2. Identificar onde o retry button dispara a operacao
3. Identificar onde a UI da fila e renderizada

**Deliverable:** Entender o fluxo atual para identificar onde esta o bug

---

### Task 2: Corrigir auto-retry ao iniciar (REQ-01)

**Action:**
1. Encontrar onde o app inicializa a fila de sync
2. Adicionar logica para processar itens pendentes automaticamente
3. Executar retry para cada item pendente
4. Atualizar status do item apos sucesso/falha

**Acceptance criteria:**
- Ao iniciar app, itens pendentes sao processados automaticamente
- Status e atualizado apos sucesso ou falha

---

### Task 3: Corrigir botao retry (REQ-02)

**Action:**
1. Encontrar callback do botao "tentar novamente"
2. Adicionar loading indicator na linha do item
3. Executar operacao de sync
4. Atualizar UI com feedback de sucesso/erro

**Acceptance criteria:**
- Loading aparece na linha do item durante tentativa
- Feedback de sucesso: item sai da fila
- Feedback de erro: mensagem de erro exibida

---

### Task 4: Adicionar ID da RAT na fila (REQ-03)

**Action:**
1. Obter ID/numero da RAT do item na fila
2. Buscar nome do cliente asociado
3. Exibir na UI: "RAT #123 - Cliente X"

**Acceptance criteria:**
- Fila mostra identificacao do RAT
- Formato: "RAT #numero - cliente" ou similar

---

### Task 5: Testar cenarios

**Action:**
1. Testar offline: salvar RAT sem rede
2. Testar recovery: reconnectar rede
3. Testar retry manual
4. Verificar que fila mostra ID correto

**Acceptance criteria:**
- RAT salva offline fica pendente
- Ao reconnectar, RAT e syncada automaticamente
- Retry manual funciona
- Fila mostra ID da RAT
