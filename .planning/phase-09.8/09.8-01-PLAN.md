# Plan 01: Error Handling Improvements

## Objective

Corrigir silent catch blocks em todo o codebase, adicionando logging e mensagens de erro apropriadas para facilitar debugging e melhorar experiência do usuário.

## Tasks

### Task 1: bootstrap.dart - Adicionar logging (EH-01)

**Action:**
1. Ler arquivo `lib/app/bootstrap/bootstrap.dart`
2. Em cada `catch (_)`:
   - Substituir por `catch (e, st)`
   - Adicionar `debugPrint('Error: $e\n$st')` ou similar
   - Manter tratamento de erro existente
3. Garantir que erros críticos ainda lancem exceptions se necessário

**Files to modify:**
- `lib/app/bootstrap/bootstrap.dart`

**Acceptance criteria:**
- Nenhum `catch (_)` sem logging em bootstrap.dart
- App continua funcionando normalmente
