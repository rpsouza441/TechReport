# Plan 01: Security Improvements

## Objective

Melhorar segurança do app corrigindo issues de autenticação, armazenamento de secrets e criptografia de backups.

## Tasks

### Task 1: supabase_client_factory.dart - Verificar PKCE (SEC-01)

**Action:**
1. Ler arquivo `lib/features/company_auth/data/services/supabase_client_factory.dart`
2. Verificar versão do `supabase_flutter` no pubspec.yaml
3. Verificar se `AuthFlowType.pkce` está disponível
4. Se disponível, migrar de `implicit` para `pkce`
5. Testar fluxo de autenticação

**Files to modify:**
- `lib/features/company_auth/data/services/supabase_client_factory.dart`

**Acceptance criteria:**
- SDK suporta PKCE: migrar para `AuthFlowType.pkce`

- Auth continua funcionando

**Acceptance criteria:**

### Task 4: flutter_secure_token_store.dart - Configurar platform options (SEC-04)
