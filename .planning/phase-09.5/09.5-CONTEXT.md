<decisions>
## Implementation Decisions

### CR-01: Error handling approach
- **D-01:** Wrap `save()` call in try-catch block
- **D-02:** On exception: call rollback logic AND set `_errorMessage` with localized message
- **D-03:** Rollback method receives previous values as parameters (not relying on saved state)
- **D-04:** Error message: "Erro ao reabrir RAT para correção."

### WR-01: UpdateTecnicoLocal injection
- **D-05:** Create `UpdateTecnicoLocal` instance once in `initState()` and store as field
- **D-06:** Reuse same instance for all save operations
- **D-07:** Alternative: inject via constructor if ViewModel uses dependency injection

### WR-02: shouldRepaint optimization
- **D-08:** Compare `strokes.length` first (fastest check)
- **D-09:** If lengths match, compare each stroke's `length` property
- **D-10:** Return `true` only if any difference found
- **D-11:** No deep comparison of stroke points (not needed, overkill)

### WR-03: Constant location
- **D-12:** Define `maxSignatureBytes` as `static const` on `RatFormViewModel` class
- **D-13:** Access via `RatFormViewModel.maxSignatureBytes`
- **D-14:** Document the value: `// 1 MB maximum signature size`

### IN-01: Comment language
- **D-15:** Replace "bytes伪造" with "bytes simulados" in test file
- **D-16:** Keep comment in Portuguese only

### IN-02: Signature ID generation
- **D-17:** Decision: Keep `microsecondsSinceEpoch` (risco muito baixo na prática)
- **D-18:** Rationale: App é single-user, microsecond precision é suficiente
- **D-19:** Document this decision in code comment for future reference

### IN-03: Fire-and-forget documentation
- **D-20:** Use `unawaited()` from `dart:async` to wrap the delete call
- **D-21:** This makes the fire-and-forget behavior explicit and idiomatic
- **D-22:** Alternative acceptable: add `// ignore: unused_result` with explanatory comment

### Claude's Discretion
- IN-02: Optamos por manter microseconds (risco prático muito baixo)
- IN-03: Preferência por `unawaited()` (mais explícito que comentário)

</decisions>
